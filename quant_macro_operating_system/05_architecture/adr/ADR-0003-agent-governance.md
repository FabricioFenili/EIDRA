# ADR-0002 — Use institutional multi-agent governance model

## Status
Accepted

## Context
The repository is designed as a governed factory rather than a loose collection of scripts. Organizational accountability and explicit routing are part of the operating model.

## Decision
Use the institutional multi-agent governance structure documented across governance, constitutions, and VP layers.

## Consequences
Architecture, responsibilities, and review cadence remain explicit. The repository retains a durable institutional structure as implementation expands.

# ADR-0005 — Use Explicit Data Contracts and Data Platform Standards

## Status
Accepted

## Context
The system requires strong dataset discipline to avoid fragile pipelines and semantic drift.

## Decision
Adopt root-level `DATA_CONTRACTS.md` and platform-level specs in `07_data_platform/` as mandatory engineering references.

## Consequences
Benefits:
- clearer schema governance
- easier onboarding
- lower data platform entropy

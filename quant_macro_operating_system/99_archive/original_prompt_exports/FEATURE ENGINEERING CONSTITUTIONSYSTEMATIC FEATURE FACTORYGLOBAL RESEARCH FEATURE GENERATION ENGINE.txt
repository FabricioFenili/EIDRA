FEATURE DEFINITION

feature = transformação determinística aplicada a dados históricos capaz de capturar estrutura informacional potencialmente preditiva.

Formalmente:

feature_t = f(data_{t−k ... t})

onde:

f = transformação definida  
data = série observada

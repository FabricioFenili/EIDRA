# Organizational Layers

Hierarquia institucional do sistema:

constituição soberana
→ VP
→ diretoria
→ superintendência
→ micro-compound
→ código / contrato / teste / ledger

Regra:
superintendências só existem quando houver:
- input próprio
- output próprio
- owner lógico
- consumer claro
- papel real no pipeline

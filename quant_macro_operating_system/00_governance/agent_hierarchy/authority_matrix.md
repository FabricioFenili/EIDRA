# Authority Matrix

| Level | Function | Authority |
|---|---|---|
| VP | functional governance | classify demand, arbitrate conflict, approve direction |
| Director | functional block | define technical line and acceptance criteria |
| Superintendent | specialization lane | define subdomain and execution path |
| Manager | workflow owner | convert demand into ordered work packages |
| Coordinator | orchestration | dispatch, merge and checkpoint micro-compounds |
| Specialist | technical executor | produce, validate and document artifact |

# Knowledge Pillars

This directory defines the scientific, technical, and pragmatic canon recommended for the executive and VP layers of the Quant Macro Operating System.

## Design principles
- high signal over volume
- foundational before fashionable
- papers and books with durable explanatory power
- direct mapping from organizational mandate to literature
- separation between executive canon, VP canon, and implementation canon

## Contents
- knowledge_pillar_governance.md
- executive_canon_ceo_council.md
- vp_literature_matrix.md
- vp_*_canon.md files for each major VP

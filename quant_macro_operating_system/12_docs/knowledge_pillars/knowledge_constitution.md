# Knowledge Constitution

## Purpose
Establish the institutional rules for how scientific, technical, and pragmatic knowledge enters, evolves, and governs the Quant Macro Operating System.

## Constitutional principles
1. Foundational before fashionable.
2. Primary sources before commentary.
3. Durable explanatory power before trend relevance.
4. Scientific validity before rhetorical appeal.
5. Operational usefulness before theoretical ornament.
6. All knowledge must be attributable to a role, mandate, and implementation surface.

## Binding hierarchy
1. Sovereign constitutions
2. VP canons
3. Directorate knowledge maps
4. Superintendency playbooks
5. Manager and specialist execution references

## Canon categories
- scientific canon
- technical canon
- pragmatic canon
- anti-pattern canon

## Replacement rule
Temporary model prior may be used only until official internal reference material is loaded.
Once an official packet is loaded, it supersedes the temporary prior without changing interfaces or routing logic.

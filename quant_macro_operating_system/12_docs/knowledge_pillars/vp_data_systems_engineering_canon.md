# VP Canon — Data Systems Engineering / Platform

## Mandate
End-to-end data platform architecture, contracts, orchestration, metadata, serving, observability.

## Core canon
1. Designing Data-Intensive Applications — Martin Kleppmann
2. Fundamentals of Data Engineering — Joe Reis, Matt Housley
3. Apache Airflow official documentation
4. Continuous Delivery — Jez Humble, David Farley
5. Refactoring — Martin Fowler

## Why these matter
- DDIA is the architectural backbone for data-intensive systems.
- Fundamentals of Data Engineering aligns with the lifecycle view of industrial data work.
- Airflow provides concrete orchestration concepts already aligned with the platform.
- Continuous Delivery and Refactoring protect long-horizon evolvability.

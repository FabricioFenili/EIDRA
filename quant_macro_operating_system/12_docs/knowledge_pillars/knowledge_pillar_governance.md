# Knowledge Pillar Governance

## Objective
Create a durable institutional canon for CEOs, VPs, directors, and specialists.

## Selection rules
1. Prefer primary sources, classic textbooks, and field-defining papers.
2. Prefer literature with durable methodological value.
3. Avoid novelty-chasing and low-signal recommendation dumping.
4. Distinguish:
   - scientific foundations
   - technical implementation foundations
   - pragmatic execution references
5. Every VP canon must contain:
   - Core Canon
   - Operating Canon
   - Pragmatic Canon
   - Exclusions and anti-patterns
   - Development priorities

## Update cadence
- annual strategic review
- quarterly tactical review only for tooling and implementation references

Support the CEO / council layer with a trans-domain canon for governing the whole system.

4. Asset Pricing — John H. Cochrane
5. Designing Data-Intensive Applications — Martin Kleppmann

## Executive operating canon
1. Fundamentals of Data Engineering — Joe Reis, Matt Housley
2. Apache Airflow official documentation

## Council directives
- CEOs should use this canon to arbitrate tradeoffs between:

  - system reliability
  - speed of implementation
  - survivability under uncertainty

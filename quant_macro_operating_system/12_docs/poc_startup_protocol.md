# POC Startup Protocol

## Goal
Allow the repository to be started in PyCharm or from CLI and complete a fake-source proof-of-concept end-to-end.

## Startup checklist
- install dependencies
- configure environment paths
- start Streamlit UI
- seed governance defaults
- seed a fake source
- create pipeline
- create source instance
- preview runtime plan
- execute E2E
- verify Platinum artifact

## Success condition
The system is considered operational in pre-setup when the chain completes without missing methods, even if the underlying business logic is placeholder-only.

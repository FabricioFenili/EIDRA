# Data Source Onboarding

## Supported source categories
- api
- file
- manual
- database

## Minimal onboarding process
1. register source name
2. define source type
3. persist source configuration JSON
4. map source to one or more pipelines
5. execute pipeline runs through the Control Plane

## Example use cases
- Bacen time series
- B3 market files
- CVM open data
- Yahoo Finance
- locally curated spreadsheets
- manual analyst inputs
